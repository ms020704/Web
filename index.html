📌 프로젝트 개요

이 프로젝트는 HTML과 CSS만으로 구현한 반응형 팀 홈페이지입니다. 팀원 소개, 기술 스택, 구현 기능 등을 포함하며 모바일에서도 최적화된 뷰를 제공합니다.

🛠️ 기술 소개

  • HTML5: 전체 구조 및 시맨틱 마크업 작성
  • CSS3: 스타일링, 레이아웃 및 반응형 구현
  • Media Queries: 반응형 웹 구현
  • Flexbox/Grid: 레이아웃 정렬과 구성

✨ 기능 소개

  • 반응형 레이아웃: 다양한 화면에서 최적의 사용자 경험 제공
  • 네비게이션 바: 메뉴 클릭 시 각 섹션으로 이동
  • 팀원 포트폴리오 연결: 팀원별 링크 제공
  • 생성성 AI : 손쉬운 개발 제공
