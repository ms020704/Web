네비게이션 바 (Navbar)

웹사이트 간편 탐색을 위한 핵심 UI 요소

홈 팀 소개 반응형 레이아웃 네비게이션 바 팀원 포트폴리오 생성형 AI

📌 네비게이션 바란?

네비게이션 바(Navigation Bar)는 웹사이트에서 사용자가 각 페이지나 섹션으로 **이동할 수 있게 해주는 메뉴 영역**입니다. 사이트 구조를 직관적으로 파악할 수 있도록 하며, 사용자 경험(UX)을 향상시키는 핵심 요소입니다.

🛠️ 기본 구조


<nav>
  <a href="#home">홈</a>
  <a href="#about">소개</a>
  <a href="#contact">문의</a>
</nav>

🎨 스타일링 예시


nav {
  background: #1f3a5f;
  display: flex;
  justify-content: center;
}

nav a {
  color: white;
  padding: 1rem;
  text-decoration: none;
}

nav a:hover {
  background: #416788;
}

📱 반응형 네비게이션

화면 너비가 좁아질 경우, 네비게이션 바가 세로 방향으로 정렬되도록 설정할 수 있습니다: 


@media (max-width: 768px) {
  nav {
    flex-direction: column;
  }
}

✅ 우리 프로젝트 적용 모습

  • 사이트 모든 페이지 상단에 고정된 nav 메뉴 구현
  • CSS Flexbox를 활용한 가로 정렬
  • Media Query를 통한 반응형 구성
  • 마우스 오버 시 배경색 변경 효과 추가
