📱 반응형 웹이란?

반응형 웹(Responsive Web Design)은 다양한 디바이스 크기(데스크탑, 태블릿, 모바일)에 따라 자동으로 레이아웃이 조절되도록 설계된 웹사이트를 의미합니다. 사용자가 어떤 기기를 사용하든지 최적화된 화면을 제공합니다.

💡 주요 구현 기술

  • Media Queries를 이용한 화면 크기 감지
  • Flexbox 또는 Grid로 유동적인 레이아웃 구성
  • 퍼센트 단위 또는 vw, vh 단위 활용

🔧 예시 코드


.container {
  display: flex;
  flex-wrap: wrap;
}

.item {
  flex: 1 1 30%;
}

@media (max-width: 768px) {
  .item {
    flex: 1 1 100%;
  }
}
