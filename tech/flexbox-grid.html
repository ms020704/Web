Flexbox & CSS Grid 소개

현대 웹 레이아웃을 위한 두 가지 핵심 기술

홈 팀 소개 HTML CSS Media Query Flex/Grid

📦 Flexbox란?

Flexbox(Flexible Box)는 1차원(가로 또는 세로) 레이아웃을 쉽게 구성할 수 있는 CSS3의 레이아웃 모델입니다. 요소들을 정렬, 간격 배치, 정렬 정돈 등에 매우 효과적입니다.

🔧 기본 예제


.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

🧩 주요 속성

  • display: flex – 플렉스 컨테이너 선언
  • flex-direction: row, column 방향 설정
  • justify-content: 가로 정렬 (center, space-between 등)
  • align-items: 세로 정렬
  • flex: 유연한 크기 조정

🔲 CSS Grid란?

Grid는 2차원 레이아웃(행과 열)을 구성할 수 있는 강력한 CSS 시스템입니다. 복잡한 페이지 레이아웃도 코드 몇 줄로 구조화할 수 있습니다.

🔧 기본 예제


.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

🧩 주요 속성

  • display: grid – 그리드 컨테이너 선언
  • grid-template-columns: 열 구성
  • grid-template-rows: 행 구성
  • gap: 아이템 간 간격
  • grid-column / grid-row: 위치 설정

💬 언제 어떤 것을 쓸까?

  • Flexbox: 메뉴, 버튼 배치 등 1차원 구조에 적합
  • Grid: 카드, 갤러리, 전체 페이지 레이아웃 등 2차원 구조에 적합
