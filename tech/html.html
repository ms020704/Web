🔎 HTML이란?

HTML(HyperText Markup Language)은 웹 페이지의 구조를 만드는 마크업 언어입니다. 문단, 제목, 링크, 이미지, 표 등을 구성하는 데 사용되며 웹의 뼈대를 형성합니다.

🧱 기본 구조


<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
  </head>
  <body>
    <h1>제목</h1>
    <p>문단 텍스트</p>
  </body>
</html>

📌 주요 태그 설명

  • <h1>~<h6> : 제목을 나타냄 (숫자가 작을수록 크고 중요)
  • <p> : 문단 단위의 텍스트
  • <a href> : 하이퍼링크 생성
  • <img src> : 이미지 삽입
  • <div>, <section> : 콘텐츠 구분용 블록

🌐 시맨틱 태그

HTML5는 웹 접근성과 구조적 의미를 높이기 위해 시맨틱 태그를 도입했습니다. 예) <header>, <nav>, <section>, <article>, <footer>
