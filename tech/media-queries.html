📱 Media Queries란?

Media Query는 디바이스의 너비, 높이, 해상도 등에 따라 웹 페이지의 스타일을 동적으로 변경할 수 있도록 하는 **CSS 기술**입니다. 즉, 하나의 웹페이지로 데스크탑, 태블릿, 모바일 등 다양한 화면에 맞게 **반응형 웹을 구현**할 수 있습니다.

💡 기본 문법


@media (조건) {
  CSS 코드
}

예제:


/* 기본 스타일 (데스크탑) */
body {
  font-size: 18px;
}

/* 768px 이하인 경우 (모바일) */
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

🛠️ 주요 조건들

  • max-width: 최대 너비
  • min-width: 최소 너비
  • orientation: 화면 방향 (세로/가로)
  • resolution: 화면 해상도

🖥️ 실제 적용 예시

네비게이션 바를 세로 방향으로 변경하는 예:


nav {
  display: flex;
  flex-direction: row;
}

@media (max-width: 768px) {
  nav {
    flex-direction: column;
  }
}
