🎨 CSS란?

CSS(Cascading Style Sheets)는 HTML로 구성된 웹 페이지에 **색상, 폰트, 레이아웃 등 시각적 스타일을 입히는 기술**입니다. HTML이 골격이라면, CSS는 그 위에 입혀지는 옷이라고 볼 수 있어요.

🧩 CSS 기본 문법


선택자 {
  속성: 값;
}
예:
p {
  color: blue;
  font-size: 16px;
}

💡 자주 쓰는 선택자

  • p, div, h1: 태그 선택자
  • .class: 클래스 선택자
  • #id: 아이디 선택자
  • div > p: 자식 선택자
  • a:hover: 가상 클래스 선택자

🔧 주요 속성들

  • color: 글자 색상
  • background-color: 배경 색상
  • margin / padding: 여백 설정
  • border: 테두리 설정
  • font-family: 글꼴
  • display, position: 레이아웃 구성

📱 반응형을 위한 Media Query

CSS에서는 다양한 화면 크기에 맞춰 레이아웃을 조정할 수 있도록 media query를 지원합니다. 


@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}
